### 2. L11 ~ L13 각 강의 섹션

각 강의 섹션 필수 블록:
1. **직관적 이해** (.tag-intuition)
2. **핵심 개념 + 공식** (.tag-theory) — MathJax 수식
3. **수치 예제** (.tag-example) — 중간 단계 전부
4. **시험 포인트** (.tag-exam) — 암기 필수 공식
5. **이 강의 기출문제** — `<div class="inline-items" data-lecture="lecXX" data-type="exam">`
6. **MMDS 관련 예제** — `<div class="inline-items" data-lecture="lecXX" data-type="mmds">`

#### L12 필수 항목 (신규 범위)
- Jaccard / Cosine(blank=0 문제) / Pearson correlation / Normalized cosine 비교
- User-User CF 예측 공식 + 수치 예제 (utility matrix)
- Item-Item CF + User-User 비교 (item-item이 더 안정적인 이유)
- CF 장단점: cold start(user/item 구분), sparsity, popularity bias

#### L13 필수 항목 (신규 범위)
- SGD 정확한 업데이트 수식 (ε_xi, u_x/v_i with 정규화)
- Regularization 목적함수 (λ₁/λ₂)
- GD vs SGD vs Mini-batch 비교
- Model with biases: r̂_xi = μ + b_x + b_i + u_x^T v_i
- Train/Validation/Test split + k 선택 곡선
- Implicit feedback: Sigmoid, BCE, BPR 공식
- GD 수렴 예제 (3~4 iteration, SSE 추적 테이블)

#### 코드/슈도코드 슬라이드 특별 처리
- L13 13-recsys-2.pdf p26: SGD for UV 슈도코드 반드시 포함
- 각 줄 한국어 주석 + "⚠️ 빈칸 출제 유력" 표시

### 5. 🔧 알고리즘 레시피 섹션
각 알고리즘 단계 순서 암기용 카드.

### 6. 📝 기출문제 전체 섹션 (id="past-exams")
- 필터 UI: 연도(전체/2024/2025) × 강의(전체/L2~L13) × 유형(전체/T·F/계산/증명)

- 각 카드: 연도·번호·유형 태그 + 문제 원문 + [풀이 보기] 토글

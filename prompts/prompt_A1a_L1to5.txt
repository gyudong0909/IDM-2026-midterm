## 파일 출력 (이 프롬프트 한 번에)

완성된 내용을 `/mnt/d/SNU/26-1/DM/A1a.html` 파일로 저장하라.
채팅창에 텍스트로 출력하지 말고, Write 도구로 파일에 직접 써라.
**완결된 standalone HTML** — `<!DOCTYPE html>`로 시작해 `</body></html>`로 닫힌 단독 실행 가능한 파일.
파일 저장 완료 후 "A1a.html 저장 완료" 메시지만 출력하고 대화를 종료하라(exit).

### 1. HTML 전체 헤더
```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>IDM 중간고사 완전정복 | 2026-04-23</title>
  <!-- MathJax CDN -->
  <script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js"></script>
  <style>
    /* 전체 CSS — 외부 파일 없음, 단일 HTML */
    /* .tag-intuition(노랑), .tag-theory(파랑), .tag-example(초록), .tag-exam(빨강) */
    /* 좌측 고정 사이드바, 스크롤 연동 active */
  </style>
</head>
<body>
```

### 4. 사이드바 + 네비게이션
링크: 기출 메타 분석 | L1~L13 | T/F 지뢰밭 | 수식 마스터 | 알고리즘 레시피 | 📝 기출 전체 | 📖 MMDS 예제 | 📋 과제 해설

### 5. 기출 메타 분석 섹션
- 2024 vs 2025 출제 패턴 비교 테이블
- 2026 예상 포인트 (Closed-book 변경 반영)
- Novel Variant 출제 이력 (A-CURE, Dω)

### 6. L1 ~ L5 각 강의 섹션

각 강의 섹션 필수 블록:
1. **직관적 이해** (.tag-intuition) — 교수 발언 인용. 왜 이 알고리즘인가.
2. **핵심 개념 + 공식** (.tag-theory) — 정확한 수식(MathJax), 의사코드
3. **수치 예제** (.tag-example) — 중간 단계 전부
4. **시험 포인트** (.tag-exam) — 암기 필수 공식, T/F 함정, Novel Variant 포인트
5. **이 강의 기출문제** — `<div class="inline-items" data-lecture="lecXX" data-type="exam">`
6. **MMDS 관련 예제** — `<div class="inline-items" data-lecture="lecXX" data-type="mmds">`

#### 코드/슈도코드 슬라이드 특별 처리
슬라이드에서 코드·슈도코드 발견 시:
- 코드 원문 수록 (들여쓰기 유지)
- 각 줄 한국어 주석
- `.tag-exam` 블록: "⚠️ 빈칸 출제 유력" + 빈칸 유력 줄 명시
- 확인된 코드 슬라이드: L5 5-itemsets-2.pdf p34 (SON MapReduce)

  /mnt/d/SNU/26-1/DM/A1a.html  ← L1~L5 + pastExams_a[] + mmdsExamples_early[]
  /mnt/d/SNU/26-1/DM/A1b.html  ← L6~L10 + pastExams_mid[] + mmdsExamples_mid[]
  /mnt/d/SNU/26-1/DM/A1c.html  ← L11~L13 + 집계 섹션 + pastExams_c[] + mmdsExamples_late[]

## 작업 지시

세 파일을 읽어 하나의 완결된 `study_guide.html`로 병합하라.
완성된 파일을 `/mnt/d/SNU/26-1/DM/study_guide.html`에 저장하라.

### 병합 규칙

#### 1. 기본 구조 (A1a.html 기준)
- `<!DOCTYPE html>`, `<html>`, `<head>`, CSS, MathJax CDN 등은 A1a.html 것을 사용
- A1b·A1c의 `<head>` 안 CSS 중 A1a에 없는 것만 추가 병합

#### 2. JS 데이터 병합
각 파일의 변수명에 접미사가 붙어 있으므로 그대로 추출 후 최종 통합 변수로 합친다:
```javascript
// A1a에서 그대로 추출
const pastExams_a = [ /* A1a.html의 pastExams_a 내용 */ ];
const mmdsExamples_early = [ /* A1a.html의 mmdsExamples_early 내용 */ ];

// A1b에서 그대로 추출
const pastExams_mid = [ /* A1b.html의 pastExams_mid 내용 */ ];
const mmdsExamples_mid = [ /* A1b.html의 mmdsExamples_mid 내용 */ ];

// A1c에서 그대로 추출
const pastExams_c = [ /* A1c.html의 pastExams_c 내용 */ ];
const mmdsExamples_late = [ /* A1c.html의 mmdsExamples_late 내용 */ ];

// 최종 통합 — pastExams_c를 권위 있는 소스로 사용 (전체 기출 포함)
// pastExams_a와 pastExams_c는 동일하므로 중복 없이 pastExams_c 사용
const pastExams = pastExams_c;

const mmdsExamples = [
  ...mmdsExamples_early,
  ...mmdsExamples_mid,
  ...mmdsExamples_late
];
```
- `renderInlineItems()`는 `pastExams` + `mmdsExamples` 전체를 사용하도록 A1c 버전 기준으로 통합

#### 3. HTML body 섹션 병합 순서
1. 사이드바 (A1c 버전 기준 — 전체 L1~L13 + 집계 섹션 링크 포함)
2. A1a의 body 섹션: 기출 메타 분석 + L1~L5
3. A1b의 body 섹션: L6~L10
4. A1c의 body 섹션: L11~L13 + T/F 지뢰밭 + 수식 마스터 + 알고리즘 레시피 + 기출 전체 + MMDS 예제 전체
   ※ 과제 해설 섹션은 B3에서 추가 — 이 단계에서 생성 불필요

#### 4. 중복 제거
- A1b·A1c의 `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>` 태그 제거 (이미 A1a에서 열림)
- A1b·A1c의 닫는 `</script>`, `</body>`, `</html>` 제거 (A1c 것만 최종에 사용)
- 사이드바 중복 제거 (A1c 것 하나만 사용)

#### 5. 최종 닫기 + B 시리즈 앵커 삽입
`</body>` 직전에 B 시리즈 삽입용 앵커 마커를 반드시 넣는다:
```javascript
  document.addEventListener('DOMContentLoaded', function() {
    renderInlineItems();
    // 사이드바 스크롤 연동 초기화
  });
</script>
<!-- B_SERIES_ANCHOR -->
</body>
</html>
```
※ `<!-- B_SERIES_ANCHOR -->`는 B1/B2/B3가 섹션을 삽입할 위치 마커. 삭제 금지.
